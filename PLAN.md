# 수강생 전용 모바일 학습 플랫폼 PRD

## 1. 서비스 개요
- 목표: 수강생들이 스마트폰으로 접속해 공지를 확인하고, 영상을 복습하며, 모바일에 최적화된 자료를 학습할 수 있는 폐쇄형 웹 앱.
- 보안: 사이트 접속 시 '공통 비밀번호(예: 0000)'를 입력해야만 메인 화면 진입 가능.

## 2. 페이지별 핵심 기능 (UI/UX)
- **잠금 화면 (`/login`)**: 공통 비밀번호 입력창. 비밀번호 미입력 시 다른 페이지 접근 불가.
- **홈 화면 (`/`)**: 로고 및 환영 인사, 최근 공지사항 노출, 최신 학습 자료 링크 카드.
- **영상실 (`/video`)**: [전체/문학/비문학] 탭 필터링, 유튜브 임베드 비디오 카드 목록 (최신순).
- **자료실 (`/material`)**: [전체/문학/비문학] 탭 필터링, 모바일에서 바로 읽을 수 있는(웹 뷰어 방식) 이미지/텍스트 기반 자료 상세 보기. 다운로드 지양.
- **공통**: 하단 고정 네비게이션 바(Bottom Navigation).

## 3. 데이터베이스 설계 (Supabase)
- `announcements`: id, content(내용), created_at
- `videos`: id, title(강의 제목), video_url(유튜브 링크), category(문학/비문학), created_at
- `materials`: id, title(자료 제목), content(자료 내용-이미지/텍스트), category, created_at
